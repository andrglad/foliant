Application Security Metrics

Collecting data and monitoring metrics is an essential part of the secure application development process.

Maverix implements data collection and consolidation mechanisms for automated metrics generation in various domains. This allows you to efficiently analyze and manage application development. The system stores and retrieves metrics data from the data warehouse.

Select the Metrics menu item at the top left of the Metrics user interface. The Metrics page appears.

The following metrics are available on the Metrics page:

  • Codebase Structure.
  • Defect Summary.
  • Scan Results.
  • Risk Density.
  • SAST Issues Discovery.
  • SCA Security Issues Discovery.
  • SCA Compliance Issues Discovery.
  • Correlation Flow.

For each metric, the user can form a narrower sample of data using a set of filters and settings.
