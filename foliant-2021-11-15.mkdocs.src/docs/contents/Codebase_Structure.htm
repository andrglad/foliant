Codebase Structure

The Codebase Structure page shows source code metrics for all Maverix applications, including:

  • Source code size by applications (measured in lines of code or number of files).
  • Source code size by programming languages (measured in lines of code or numbers of files).

 

 

To get the code profile for a particular application(s) or/and programming language(s), select the application(s) from the drop-down menu in the Applications field or/and the programming language(s) from the drop-down menu in the Languages field on the Filters tab at the top right.

Click the button at the top right to reset the filters.
