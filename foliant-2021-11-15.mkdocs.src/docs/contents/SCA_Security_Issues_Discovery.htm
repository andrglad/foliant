SCA Security Issues Discovery

The SCA Security issues discovery page shows the following metrics:

  • SCA vulnerabilities grouped by applications if the Applications item is chosen on the Group by tab, distributed by severity.
  • Security vulnerabilities grouped by components if the Components item is chosen on the Group by tab, distributed by severity.

 

 

To get the SCA Security issues discovery metrics for a particular application(s), select the Applications item on the Group by tab and select the application(s) from the drop-down menu in the Applications field on the Filters tab at the top right.

To get the SCA Security issues discovery metrics for a particular component(s), select the Components item on the Group by tab and select the component(s) from the drop-down menu on the Components field on the Filters tab.

Use the drop-down menus in the Severity and Status fields on the Filters tab to get the metrics for SCA Security vulnerabilities with a particular severity and/or status.

Click the button at the top right to reset filters.
