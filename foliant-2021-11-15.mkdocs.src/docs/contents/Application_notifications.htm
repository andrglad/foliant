Application notifications

The Notifications menu item allows you to configure the sending of notifications about various events related to the application. Notifications are sent by email.

It’s important to emphasize the settings of notifications are available both for each application and for the user. For instance, if the notifications on the reopened defects are enabled in the application settings (see below), but these are disabled in the user profile (for more information, see item "User Profile Setup" of Application Administrator Guide), the mentioned notifications won’t be sent to the user.
