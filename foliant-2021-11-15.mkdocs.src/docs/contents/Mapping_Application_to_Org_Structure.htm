Mapping Application to Org Structure

Select the Team tab on the application Settings page.

 

 

The current position of the application in the company organizational structure is shown in the Org structure section.

Update the application organizational structure using the Change button on the right. The Assign application to another unit window appears.

 

 

Click the current org structure unit, choose the new org unit from the list and click the Assign to selected unit button

 

 

 

and confirm the org structure update in the appeared confirmation window. The Org structure section shows the updated data.

 

 

The Team tab allows specifying information about team size and maturity level in the Team size and Team Maturity fields. This information is optional, so these fields can be left blank.
