Export to CI/CD

Once the security pipeline is created and configured in Maverix, it can be exported to the CI/CD tool.

 

If any changes have been performed to the security pipeline in the Maverix user interface, it's necessary to re-export parameters to CI/CD tools. Please see below.

 

Click the Actions  button at the top right of the Structure tab and select the Export CI/CD item from the drop-down menu to export the security pipeline to the orchestration tool (e.g., TeamCity).

 

 

The confirmation notification appears at the bottom right of the Maverix user interface.

 

 

As a result, a new Security Pipeline will be created in TeamCity with configuration parameters defined in Maverix. Now TeamCity can run the Security Pipeline configured in Maverix. TeamCity runs all the tools in the pipeline and provides security testing results to Maverix.

Click the Export CI/CD button at the bottom right of the security pipeline card on the DevSecOps page to export the security pipeline to the CI/CD tool.
