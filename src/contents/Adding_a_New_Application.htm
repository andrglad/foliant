Adding a New Application

Select the Applications menu item at the top left of the Maverix user interface and click the +Add new button to create a new application.

 

 

Specify the Application name, Workspace, and Organization structure parameters for the new application using drop-down menus in the Create application window. The workspace and organizational unit can be selected from the list available to the manager's team(s). Administrator can assign Manager to work with more teams before an application is created. In this case, more workspaces will be available for the application being created.

 

 

If the application creation process was successful, the confirmation message will appear at the bottom right of the Applications page.
