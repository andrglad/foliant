Parameters for interaction with Checkmarx

Parameter Required parameter Description Example
--cx-tool-url + URL of the Checkmarx instance --cx-tool-url https://cx.dev.maverix.co
--cx-project-name + The name of the project in the Checkmarx instance above --cx-project-name DVJA_dvja-master
--cx-team – Checkmarx team name, by default: CxServer --cx-team CxServer\DEV\DVJA
--cx-scan-id – Scan ID in the Checkmarx project. If this parameter is absent, the results of the last scan are imported --cx-scan-id 1000650
