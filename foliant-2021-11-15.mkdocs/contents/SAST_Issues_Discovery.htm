SAST Issues Discovery

The SAST issues discovery page shows the following metrics:

  • SAST issues grouped by applications if the Applications item is chosen on the Group by tab, with severity distribution.
  • SAST issues grouped by categories if the Categories item is chosen on the Group by tab, distributed by severity.

 

 

To get the SAST issues discovery metrics for a particular application(s), select the Applications item on the Group by tab and select the application(s) from the drop-down menu in the Applications field on the Filters tab at the top right.

To get the SAST issues discovery metrics for a particular category(s), select the Categories item on the Group by tab and select the category(s) from the drop-down menu in the Categories field on the Filters tab.

Use the drop-down menus in the Severity and Status fields on the Filters tab to get the metrics for SAST issues with a particular severity and/or status.

Click the button at the top right to reset filters.
