Application Instances

Each application can have more than one instance. For example, one application can have instances for development, testing, and production.

Click the Development menu item on the left and then the +Add newbutton at the top right of the Instances tab to add a new instance or click the edit icon on the existing instance card to check/configure the instance.

 

 

The Update instance window appears.

 

 

Specify the following application instance parameters in this window:

  • Instance name
  • URL
  • Select Stage drop-down menu (System test, User Acceptance Test, Production, etc.).
  • Credentials field specifies a list of users and their passwords for accessing the application instance. Use the Add button on the right to add a new user. Click the icon next to the Credentials field to make user passwords visible. Click this icon once more to hide passwords again.

Click the Update button to save changes.
