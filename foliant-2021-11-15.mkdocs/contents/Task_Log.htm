Task Log

The Task log menu item allows you to view the log with records of various events for a selected application.

 

 

This page contains three tabs:

  • Imports — tasks related to importing security issues from other tools. The following data is displayed:
    • Id.
    • Status.
    • Started.
    • Duration.
    • Warnings. If there are warnings, an icon is displayed in this column (see the picture above). Click this icon to view the list of warnings. 


    • Scan id.
    • Pipeline — a link to the security pipeline where the scan was performed.
    • Started by — the name of the user who initiated the security issues import process. If this field is empty, an import has been started automatically, for instance, after scanning completion.
  • Scans — tasks related to security pipelines scanning.
  • Tracker syncs — tasks related to synchronization with a Defect Tracking System.
