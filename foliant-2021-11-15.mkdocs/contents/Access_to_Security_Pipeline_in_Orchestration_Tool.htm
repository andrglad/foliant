Access to Security Pipeline in Orchestration Tool

Select the Applications menu item at the top left of the Maverix user interface and select an application. Click the edit icon at the bottom right of the application card on the Applications page. Click the DevSecOps menu item on the left. Click the Show pipeline config icon at the bottom right of the security pipeline card.

Click the link to the orchestration tool to the right of the Last export CI/CD item to access the security pipeline in TeamCity.

 

 

The TeamCity login page will appear on the screen.
